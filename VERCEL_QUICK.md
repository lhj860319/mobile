# ⚡ Vercel 빠른 배포 (2분 완성)

## 🚀 3단계로 완료!

### ✅ 1단계: Vercel 접속 및 로그인

1. **https://vercel.com** 접속
2. **"Sign Up"** 또는 **"Log In"** 클릭
3. **"Continue with GitHub"** 클릭
4. GitHub 권한 승인

---

### ✅ 2단계: 프로젝트 가져오기

1. **"Add New..."** → **"Project"** 클릭
2. GitHub 저장소 목록에서 **`mobile-p`** 찾기
3. **"Import"** 버튼 클릭

---

### ✅ 3단계: 배포 설정 및 실행

**설정:**
- Framework Preset: **Other**
- Build Command: **(비워두기)**
- Output Directory: **(비워두기)**

**배포:**
- **"Deploy"** 버튼 클릭
- 1-2분 대기

---

## 🎉 완료!

배포 완료 후:
- ✅ 자동 생성된 URL 확인
- ✅ URL 클릭하여 사이트 테스트
- ✅ URL 공유

**예시 URL:**
```
https://sj-mobile-partner-center.vercel.app
```

---

## 🔄 코드 업데이트 시

```bash
git add .
git commit -m "업데이트"
git push
```

→ Vercel이 자동으로 재배포!
